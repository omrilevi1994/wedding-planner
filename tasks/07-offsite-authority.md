# 07 — Off-site authority

**Priority:** P2 · **Depends on:** 02, 06 (something worth linking to must exist first) · **Effort:** ongoing

A brand-new domain with zero backlinks won't rank against established Israeli wedding portals on content alone, and AI engines weight entity corroboration (the same product described consistently across independent surfaces). Start after the landing page ships.

## Subtasks

### 7.1 Social / profile presence (entity corroboration)
- [ ] Create profiles with the consistent entity signature (task 4.4): Instagram + Facebook (primary channels for the Israeli wedding audience), LinkedIn company page, optionally TikTok.
- [ ] Each bio links to `https://wedflow.live`; add all profile URLs to `Organization.sameAs` schema (task 3.4).

### 7.2 Directories & listings
- [ ] Israeli startup/product directories (Startup Nation Central, Geektime coverage, etc.).
- [ ] Product directories: Product Hunt launch (also a launch-marketing moment), AlternativeTo, relevant SaaS listings.
- [ ] Wedding-specific: Israeli wedding vendor/tool directories where a planning app fits.

### 7.3 Earned links (quality over volume)
- [ ] Pitch the data story: once task 5.3's budget/guest-list data content exists, pitch Israeli lifestyle/tech media — original wedding-market data is genuinely linkable.
- [ ] Partnerships with wedding vendors/venues/photographers: they recommend WedFlow to couples, WedFlow's vendor directory features them — natural reciprocal visibility (keep links editorial, not schemes).
- [ ] Answer wedding-planning questions in Israeli communities (Facebook groups, Reddit r/Israel wedding threads) where genuinely helpful — brand mentions matter for GEO even unlinked.

### 7.4 Hygiene
- [ ] Never buy links; no link farms, no mass directory spam (toxic-link risk on a young domain is fatal).
- [ ] Quarterly: check the backlink profile (Search Console links report at minimum) for toxic patterns.

## Acceptance criteria

- ≥3 live social/entity profiles, consistent signature, wired into `sameAs`.
- ≥5 legitimate directory/listing links.
- 1 data-driven PR pitch sent (after the data content exists).
- Baseline backlink snapshot recorded (Measured, dated) for tracking in task 08.
